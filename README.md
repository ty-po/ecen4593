#ECEN 4593 Final Project
Memory Simulation

##Usage
To Run Once

1. in `src/` do `cmake .`, `make`
2. in `bin/` do `cat trX | ./ecen4593final`

To Watch

1. in `test/utils/` do `./watch.sh`
2. or `./watch.sh trX
